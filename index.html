ThingsPanel 安装程序

一键部署企业级 IoT 平台

选择您的操作系统，获取安全、自动化、全封装的安装体验。

推荐 (服务器)

Linux

支持 CentOS, Ubuntu, Debian 及通过 SSH 访问的 macOS。

执行以下一键安装命令：
curl -fsSL https://install.thingspanel.io/install.sh | sh

Windows

提供带界面的绿色安装包，纯净无依赖。

下载 .exe 安装向导

macOS

原生支持 Apple Silicon / Intel，基于 Docker Desktop 分发。

下载 .pkg 安装包

安装遇到问题？请参考 文档与故障排查
